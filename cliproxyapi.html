Giới thiệu

CLIProxyAPI là proxy server viết bằng Go cực kỳ phổ biến trong cộng đồng AI (tháng 1/2026), giúp bạn sử dụng các mô hình cao cấp miễn phí hoặc dựa trên subscription cá nhân:

  • Gemini 2.5 Pro • Gemini 3 Pro
  • Claude Code/Max • Claude 3.5/4
  • GPT-5 • Codex
  • Qwen Code • Antigravity • iFlow

Chỉ cần đăng nhập 1 lần → token lưu lại → dùng thoải mái trong IDE mà không tốn thêm API key trả phí.

Cài đặt nhanh

Docker (khuyên dùng)

docker run -d -p 8317:8317 \ -v ~/.cli-proxy-api:/root/.cli-proxy-api \ -v ./config.yaml:/CLIProxyAPI/config.yaml \ eceasy/cli-proxy-api:latest

Homebrew (macOS)

brew install cliproxyapi brew services start cliproxyapi

Sau khi chạy → trỏ client đến http://localhost:8317/v1 và dùng model như claude-3.5-sonnet, gemini-2.5-pro...
